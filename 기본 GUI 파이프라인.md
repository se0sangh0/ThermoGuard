프로그램 실행
├ 환경 설정
	├ 카메라 연결 확인
		├ 현재 지정되어 있는 연결 대상 카메라 ip 정보 나와야함
		├ 카메라가 연결되었다면 연결되었다고 피드백 (비동기 검사)
		└ System Status 표시 (Not Ready / Ready / Monitoring / Error)
	├ Flow 설정
		├ Mode 선택 (thermal-only / both)
		└ ExifTool 경로 설정 (자동 감지 또는 수동 지정)
	├ 디렉토리 정보 확인
		├ 현재 지정되어 있는 디렉토리 정보
		└ 저장 디렉토리 변경 기능
	└ 저장 주기 설정
		├ 현재 지정되어 있는 저장 주기 정보
		└ 저장 주기 변경 기능 
├ 감지 화면
	├ 가장 최근 캡쳐된 이미지 노출
		└ 열화상 이미지
			└ 열화상 이미지에 온도 정보 나와야함 (Max/Mean/95th)
				└ 어느 부분에서 온도가 검출되었는지 디텍팅 되어야함 (핫스팟 마커)
	├ 가시광/열화상 전환 버튼
		└  버튼 누르면 열화상, 가시광 이미지 변경되어야함
	├ 데이터 운영 버튼
		├ Check Dataset → 무결성 검사 및 NPY 복구 (병렬)
		├ Generate Metadata → metadata.csv 생성
		└ Cleanup Dataset → 보존 기간 지난 데이터 정리
	├ ROI 설정 버튼
		├ 버튼 클릭 시 ROI 설정 화면 실행 (메인 스레드)
			├ 이전에 캘리브레이션 진행한 이력 없는 경우 캘리브레이션 설정 화면 노출
				└ 캘리브레이션용 이미지는 저장되어 있는 이미지 중 감지 열원이 가장 많은 이미지 기준
					└ 저장된 이미지 중 열원 감지된 이미지가 없는 경우 가장 최근 이미지
		└ 이 화면들은 별도의 gui 창의 띄워서 작업
			└ 화면 좌측 하단에 사용설명이 나와야함(S는 저장, Q는 종료, Z는 직전 선택값 취소 등)
└ 로그 화면	
	├ Detection Log 탭
		└ 감지일, 감지 위치, 감지 온도, 경고 단계, 알림 전송 여부
	└ Activity Log 탭
		└ 운영 로그, 오류 메시지, 백그라운드 작업 상태
